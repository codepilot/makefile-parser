makefile-parser
===============

**WORK IN PROGRESS**

A parser for [Makefiles](http://www.gnu.org/software/make/manual/make.html).

Current form of this program is in the programming language Euphoria (see rapideuphoria.com),

The current plan to it migrate this to Node JS.


The folder "preFiles" contains the contents of the <pre> tags from https://www.gnu.org/software/make/manual/make.html
